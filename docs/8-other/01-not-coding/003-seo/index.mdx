# アクセスを増やしたい

ウェブ開発において、ウェブサイトのアクセスを増やしたいとき、どのようにすれば良いでしょうか？

大きなものとして、検索流入を増やすことと SNS で拡散することがあるでしょう。

ここでは検索流入を増やす方法である「SEO 対策」を見ていきます。

# SEO とは

SEO は Search Engine Optimization の略で、「検索エンジン最適化」という意味になります。

Google 検索などの**検索エンジン**はアルゴリズムによって動いており、それこそが SEO です。

検索流入を増やすためには検索順位を上げることが重要です。
そして検索順位を上げるために SEO を対策するのが「SEO 対策」なのです。

SEO 対策には色々なものがあり、Google のガイドラインを読んだり、その他ネット上の情報を参考にすると良いでしょう。
